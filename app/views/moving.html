Servicio de Mudanzas


El servicio de mudanzas de Multicasa aún no esta disponible en todas las ciudades de la Republica Mexicana, le pedimos paciencia para poder llevar este servicio a cada ciudad, las ciudades que ya cuentan con este servicio son:


- Acapulco
- Apodaca
- Campeche
- Chihuahua
- Ciudad de México
- Ensenada
- Guadalajara
- Juarez
- Morelia
- Saltillo
- Veracruz
- Zacatecas

El servicio de mudanzas de Multicasa incluye:


- 3 cargadores
- Herramienta para maniobras
- Hasta 2 emplayes de regalo
- Maniobras de carga y descarga
- Rampa y bandas para maniobra

En Multicasa tratamos cada uno de los bienes que transportamos como si fueran nuestros.

Para solicitar el servicio de mudanza por favor manda un correo desde la pagina de contacto explicando que quieres contratar este servicio.
